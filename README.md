# Research Materials Website

Static website for presenting research assets in Hebrew (RTL):

- Poster preview
- Full survey (PDF)
- Project summary (PDF)
- Full paper (PDF)
- Additional materials / graphs (PDF)
- Contact page

## Suggested product priority

If the next decision is between **adding more pages** and **improving design quality**, prioritize:

1. **Homepage polish and visual consistency first**
2. Then expand with additional pages/content

Reason: once the visual system is stronger, every new page is faster to build and feels professional by default.

## Fast wins for professionalism

- Keep one clear typography scale for all pages (title / subtitle / body).
- Keep consistent spacing rhythm (same vertical gaps across sections).
- Ensure every interactive element has hover/focus states.
- Add short explanatory subtitles for every card (avoid empty descriptions).
- Keep strong color contrast for readability.

## Current structure

- `index.html` — homepage with quick navigation cards
- `styles.css` — shared design system for all pages
- `poster.html`, `survey.html`, `summary.html`, `paper.html`, `materials.html`, `contact.html`

All pages are intentionally lightweight and can be deployed on any static host.
